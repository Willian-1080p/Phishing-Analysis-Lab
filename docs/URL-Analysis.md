# URL Analysis

## Objetivo

Avaliar URLs suspeitas antes do acesso.

## Itens Avaliados

### Domínio

Verificar se o domínio é legítimo.

### HTTPS

Validar certificado e criptografia.

### URL Longa

URLs excessivamente longas podem indicar phishing.
