# Sandbox Reports

## Objetivo

Analisar arquivos e URLs em ambiente isolado.

## Tipos de Análise

### Estática

Sem execução do arquivo.

### Dinâmica

Execução controlada em sandbox.

## Informações Coletadas

### Processos

Processos criados pelo malware.

### Rede

Conexões externas.

### Arquivos

Arquivos criados ou modificados.

### Registro

Alterações no Windows Registry.

## Ferramentas

- Any.Run
- Hybrid Analysis
- Joe Sandbox
- Cuckoo Sandbox

## Indicadores Importantes

- Persistence
- C2 Communication
- DLL Injection
- PowerShell Abuse
- Credential Theft

## Evidências

- Hashes
- Screenshots
- Network Traffic
- Process Tree
